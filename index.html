CSS SELECTORS AND SPECIFICITY

It is the beginning of the set of rules for identifying the element to which to apply the desired style.


SELECTORS


TYPE SELECTOR corresponds to the type of element to apply the style to, it is called with the name of the element's tag or selector
UNIVERSAL SELECTOR as the name suggests select elements of any type, use * before the bracket {
CLASS SELECTOR we use a class= attribute to the HTML code, and in the CSS sheet we write a period ( . ) followed by the value given to the attribute
MULTIPLE CLASS we can add more classes by separating the attributes with a space ( .attribute1 .attribute2 )
ID SELECTOR to give an element a unique style, we give the id= attribute in our HTML sheet, while in the css sheet we put # before the name of the ID
ATTRIBUTE SELECTOR you can directly select an attribute by placing it directly between [example] or type[attribute*=value]
PSEUDO CLASS the appearance of some elements changes with user interaction. this allows a pseudo-class that can be connected to any selector. just put i : after the selector followed by the appropriate name (example: hover { )
MULTIPLE SELECTORS to add styles to multiple selectors at the same time just separate the selectors with a comma ,

SPECIFICITY


Specificity is the order in which the browser decides which CSS styles are displayed. A better practice is to style elements using the lowest degree of specificity. ID is the most specific selector, followed by classes, and finally type selectors. In this order it will be displayed and given priority by the browser during reading.
